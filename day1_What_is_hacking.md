# Day 01 — Ethical Hacking 101
[← Back to overview](README.md)

## What is "Hacking"?
- **Hacking = Curiosity.** Outsmarting a system and making it do something outside its intended purpose.
- It's a mindset/skillset, not inherently good or bad — intent and authorization make it "ethical" or "criminal."

## What is Ethical Hacking?
- The practice of **legally and intentionally** breaking into systems, networks, or apps to find and fix vulnerabilities before malicious actors exploit them.
- **Ethical Hackers / White Hats** = professionals who use hacking skills for defense, under a signed scope of engagement (SOW/contract).

## Types of Hackers
| Type | Description |
|---|---|
| **White Hat** | Authorized, works for defenders (pentesters, security researchers) |
| **Black Hat** | Unauthorized, malicious intent (theft, damage, extortion) |
| **Grey Hat** | No authorization but no malicious intent — technically illegal, ethically ambiguous |
| **Script Kiddies** | Low-skill, uses existing tools/scripts without deep understanding |
| **Hacktivists** | Hack for political/social causes (e.g., Anonymous) |
| **State-Sponsored** | Backed by governments — espionage, sabotage, cyberwarfare |

## Testing Approaches (Knowledge given to the tester)
| Testing Type | Knowledge Level | Focus Area | Common Use Case |
|---|---|---|---|
| **White Box** | Full knowledge of code/infrastructure | Internal structure & logic | Unit & integration testing |
| **Black Box** | No prior knowledge | Functionality & user experience (simulates real attacker) | System & acceptance testing |
| **Grey Box** | Partial knowledge | Both internal & external aspects | Integration & system testing |

📝 *My note:* Black box = closest to how a real external attacker sees you. White box = closest to how a malicious insider or someone who's read your source code sees you.

## Red Team vs Blue Team vs Purple Team
| Team | Focus | Key Activities | Goal |
|---|---|---|---|
| **Red Team** | Offensive security | Simulated attacks, penetration testing | Find vulnerabilities |
| **Blue Team** | Defensive security | Monitoring, incident response | Protect against attacks |
| **Purple Team** | Collaboration | Joint exercises, knowledge sharing | Strengthen overall security posture |

## ➕ Extra context (added beyond the slides)
- **Bug bounty programs** (HackerOne, Bugcrowd) are the legal, paid real-world version of grey/white-hat hacking.
- Common certifications tied to this career path: **CEH, OSCP, eJPT, CompTIA Security+, CompTIA PenTest+**.
- The **CIA Triad** (Confidentiality, Integrity, Availability) is the foundational goal security teams protect — almost every attack maps to breaking one of these three.
